## Static Website

Hosted via **GitHub Pages** at `edu.scientia-naturalis.org`

### Features
- Mobile-first, 12-column CSS Grid layout
- W3C Verifiable Credential showcase (Erasmus+ CBHE)
- Linked project/repository reference section
- Zero dependencies — pure HTML/CSS/JS

### Deploy to GitHub Pages

1. Fork or clone this repository
2. Go to **Settings → Pages**
3. Set source to `main` branch, root `/`
4. Add custom domain: `edu.scientia-naturalis.org`
5. Add a `CNAME` file containing `edu.scientia-naturalis.org`

### Adding Projects

Edit the `#projects` section in `index.html` — update the `href` on each `.repo-link` anchor to point to your real GitHub repositories.

### Credential Schema

W3C VC schemas are published in the linked repositories. The issuer DID resolves at:  
`did:web:edu.scientia-naturalis.org`
